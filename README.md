This project is made as a part of The Odin Project Curriculum.<br/>
It's a simple calculator performing basic math functions, with a restriction on typing more than 1 number after the decimal point. It's built using HTML, CSS and vanilla JavaScript.<br/>
The calculator accepts user input through a UI, performs a calculation and displays the result of it to the user.<br/>
You can try it out here: https://osechi3.github.io/calculator/
